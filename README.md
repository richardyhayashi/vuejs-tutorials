## Vue Project Setup

### Create

`$ npm create vue@latest {.|project-name}`
(Typescript, JSX Support, Vue Router, Pinia, Unit Testing, End-to-End Testing, ESLint, Dev Tools)

### Change Port

In 'vite.config.js';
`export default defineConfig({`<br>
`***`<br>
`server: {`<br>
`    host: "0.0.0.0",`<br>
`    port: 3000,`<br>
`}`<br>
`***`<br>
`})`<br>

### Run Server

`$ npm run dev`
